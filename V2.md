# Milan Remote Work Map — V2

Find cafes, restaurants, libraries, and other places where you can work remotely in Milan.

**Reference:** https://spieli.eu/ (filter UI & popover legend approach)

## Features (MVP)

**Map Interface**

- Full-screen map, centered on user location (via browser geolocation) or Milan (default fallback)
- Pins clustered at zoom < 14
- Click pin to show place details (name, address, hours, amenities)

**Legend & Filters** (popover)

- Toggle filter by: has_internet, has_sockets, open_now
- Display legend for pin colors/icons

**Pages** (collapsible menu)

- Map (default)
- How to contribute (link to OSM/MapComplete)
- About

**Data Filtering**
Only show places with `laptop=yes` in OSM. Use tags:

- `laptop` (yes/no, main filter)
- `laptop:conditional` (time/location restrictions, display as note)
- `opening_hours` (for "open_now" filter & display)
- `internet_access` (yes/no, filter)
- `sockets` (yes/no/info, filter)

(Add more tags later: wheelchair, quiet, etc.)

## Tech Stack

Typescript everywhere.

**Frontend**

- Vite + LitElement + Leaflet

**Backend** (2 components)

- **Server**: SQLite + REST API endpoint returning JSON/GeoJSON of places
- **Worker**: Downloads Geofabrik `.osm.pbf` extracts for a configured list of Italy
  regions, filters them locally with `osmium-tool`, and upserts into SQLite
  - Trigger: on-demand via `npm run worker` or scheduled daily
  - Regions are processed one at a time; `PBF_REGIONS` env var can scope a run to a subset

**Database**

- Start with plain SQLite (store lat/lon as regular columns, filter with WHERE queries)
- Future: migrate to SpatiaLite for spatial indexes & bounding-box queries if performance needs it
